Setup

source default
artboard 4" x 4"
stroke: main 24pt
stroke: auxiliary* 18pt
stroke: corner .15"
stroke: join round

NOTE: Rememeber to expand to outlines before committing.

Class Size

pi-sm font-size: .75rem
pi-lg font-size: 1.5rem
